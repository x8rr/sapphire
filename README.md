# sapphire
a fork of [carbonicality/amethyst](https://github.com/carbonicality/amethyst) that aims to improve the functionality of it

## what is amethyst?
amethyst is a chrome extension runtime for web pages, allowing chrome extensions to be usable in webpages e.g proxy sites utilising Scramjet. 

in its current state, it is a bit broken and not every works. extensions such as vencord DO NOT work yet.

it aims to create a 'compatibility layer' and emulate almost all of the chrome.* APIs.
